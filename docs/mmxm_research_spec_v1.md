# MMXM Research Specification – v1

## 1. Doel

Het doel van dit onderzoek is vaststellen welke instapmethode binnen MMXM het meest stabiel en reproduceerbaar is:
- vroeg instappen
- instappen na bevestiging

Dit onderzoek richt zich niet op maximale winst, maar op consistentie, stabiliteit en automatiseerbaarheid.

## 2. Afbakening (Optie A)

Voor deze eerste versie geldt:
- één markt
- één timeframe
- geen hogere-timeframe filters
- geen fractals
- geen subjectieve keuzes

Alleen:
- prijsdata
- vaste regels
- objectieve metingen

## 3. Marktstructuur (MMXM-kader)

De markt beweegt cyclisch door drie fases:
1. opbouw (accumulatie)
2. verstoringsfase (manipulatie / liquiditeit ophalen)
3. beweging (distributie)

Instappen kan op verschillende momenten binnen deze cyclus.

## 4. Instapmethodes

### A. Vroege instap
- instap direct bij een belangrijk prijsgebied
- geen bevestiging van richting
- gebaseerd op verwachting van omkering

Kenmerken:
- betere prijs
- hoger risico
- meer variatie in resultaat

### B. Instap na bevestiging
- instap pas nadat richting aantoonbaar is veranderd
- markt heeft laten zien dat eerdere deelnemers zijn uitgestapt

Kenmerken:
- slechtere prijs
- lager risico
- meer consistentie

## 5. Kernonderzoeksvragen (v1)

1. Welke instapmethode is consistenter?
2. Welke instapmethode heeft lagere drawdown?
3. Is vroege instap alleen effectief bij juiste context?
4. Welke methode is stabieler over tijd?
5. Welke methode is beter reproduceerbaar zonder interpretatie?

## 6. Testopzet

- beide instapmethodes worden getest op exact dezelfde data
- de enige variabele is het instapmoment
- elke trade krijgt een label:
  - “vroeg”
  - “bevestigd”

Per groep meten we:
- gemiddelde opbrengst
- winrate
- drawdown
- stabiliteit
- extreme verliezen

## 7. Beslisregels

Na de test geldt:
- als één methode duidelijk stabieler is → die wordt basis
- als beide werken → combineren in latere fase
- als vroege instap te instabiel is → alleen nog als aanvulling

Dit bepaalt hoe jouw uiteindelijke stijl wordt opgebouwd.
